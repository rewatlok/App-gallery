# Приложение Галерея Фотографий

Десктопное приложение для управления фотографиями с категориями, полноэкранным просмотром и групповыми операциями. Создано на Python (бэкенд) и QML (фронтенд) для управления фотографиями: позволяет импортировать изображения, организовывать их по категориям, просматривать в полноэкранном режиме, настраивать размер миниатюр и хранить всё в SQLite базе данных.

Взаимодействие с проектом: запустите python main.py, добавляйте фото через кнопку "Добавить фото", управляйте категориями через соответствующий интерфейс. Двойной клик открывает полноэкранный просмотр, используйте контекстное меню (правый клик) для дополнительных опций.
